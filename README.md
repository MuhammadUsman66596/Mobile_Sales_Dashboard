# Mobile_Sales_Dashboard
📊 Mobile Sales Dashboard (Power BI)

🔍 Overview

This project is an interactive Power BI dashboard designed to analyze mobile phone sales performance across different cities, brands, and time periods. It provides clear insights into sales trends, customer behavior, and transaction methods.

---

🚀 Key Features

- 📈 Total Sales Analysis (769M+)
- 📦 Total Quantity Sold (19K+ units)
- 💳 Transaction Breakdown by Payment Method
- 📅 Monthly Sales Trends
- 🌍 Sales Distribution by City (Map Visualization)
- 📱 Sales by Mobile Models
- ⭐ Customer Ratings Analysis
- 📊 Brand-wise Performance Comparison

---

🛠️ Tools & Technologies Used

- Power BI – Data visualization and dashboard creation
- Microsoft Excel / Dataset – Data source
- DAX (Data Analysis Expressions) – Calculations and measures

📊 Insights Derived

- Apple and Samsung lead in overall sales performance
- Sales peak observed in mid-year months
- UPI and Card payments dominate transaction methods
- Customer ratings are mostly positive (4–5 stars)
- Major sales contributions come from metropolitan cities

---

⚙️ How to Use

1. Download the ".pbix" file
2. Open it in Power BI Desktop
3. Refresh dataset if needed
4. Interact with filters (Month, Brand, Payment Method)
